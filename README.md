# Khanvict Command Center (KCC)

A content scheduling and posting command center built for streamers. Plan your clips, schedule posts across TikTok, Instagram Reels, and YouTube Shorts, generate AI-powered captions, and track your engagement — all from your phone.

**Built by [@khanvicttv](https://tiktok.com/@khanvicttv)**

---

## What It Does

- **Schedule posts** across TikTok, Instagram, and YouTube with a weekly calendar view
- **Today screen** — see exactly what to post and when, with one-tap caption copy
- **AI captions** — generate platform-optimized captions with trending hashtags (requires Anthropic API key)
- **Content Intel** — AI-powered weekly research that finds trending topics for your next streams
- **Engagement tracking** — log views, likes, comments and see what's working
- **Analytics dashboard** — platform comparisons, best posting times, hashtag performance
- **Push notifications** — 30-minute reminders before scheduled posts
- **Works offline** — PWA that installs on your phone like a native app

---

## Quick Start

1. **Visit the app:** [khanvicttv.github.io/kcc-public](https://khanvicttv.github.io/kcc-public)
2. **Install as app:** On iPhone/Android, tap Share → Add to Home Screen
3. **Add clips:** Go to Clips tab → tap + to add your clips with names, descriptions, and target platforms
4. **Schedule:** Drag clips onto the weekly calendar or tap a slot to assign
5. **Post day:** Open Today tab → copy captions → paste into TikTok/IG/YT → mark as posted

## Optional Setup

### AI Captions & Content Intel (free)
1. Get a free API key at [console.anthropic.com](https://console.anthropic.com)
2. In KCC → Settings → paste your Anthropic API key
3. Now you can generate captions and run Content Intel research

### Cloud Sync (free)
Sync your data across devices using GitHub:
1. Create a free [GitHub](https://github.com) account
2. Create a new repo (any name)
3. Generate a personal access token with repo permissions
4. In KCC → Settings → enter your GitHub username, repo name, and token

### Push Notifications (free)
Get reminded 30 minutes before each post:
1. Install the [ntfy app](https://ntfy.sh) on your phone
2. Pick a channel name (something unique like `yourname-kcc-xyz`)
3. In KCC → Settings → enter your channel name → tap Test

---

## Your Data

All your data stays in **your browser's local storage** — nobody else can see it. If you set up GitHub sync, your data goes to **your own private repo**. The app never sends your data anywhere else.
